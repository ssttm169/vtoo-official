Intro

Let's discover Vtoo in less than 1 minutes.

Vtoo is an advanced image batch processing app that combines high performance with a lightweight design and offline capability. It supports various formats, including JPG, PNG, GIF, and BMP. Beyond compression, vtoo offers powerful features such as resizing, format conversion, watermarking, and more.

Getting Started​

Please download application from vtoo offcial website: https://vtoo.app

Operation System requirements​

  • Windows version windows10(x64) or above:
  • MacOS comming soon

Image Compression​

This is the core feature of vtoo, allowing batch image compression by default.

  • Free users can compress up to 20 images at a time, with each image up to 8MB.
  • Pro users can compress an unlimited number of images at a time, with each image up to 80MB. Note: These limits are subject to client hardware conditions.
  • Ultra users can compress an unlimited number of images at a time, with each image up to 180MB. Note: These limits are subject to client hardware conditions.

Resizing​

This is a paid feature in vtoo that allows batch resizing of multiple images, saving time and effort. Available exclusively to Ultra users.

Format Conversion​

This is a paid feature in vtoo that enables batch conversion of multiple images.

Watermarking​

This is a paid feature in vtoo that allows adding custom text watermarks to multiple images. Available exclusively to Ultra users.
